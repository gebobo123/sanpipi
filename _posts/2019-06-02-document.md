本文为针对完全没有基础的小白。

## 说明

* 系统: Ubuntu 20.04

* x86 处理器

* nas , 云服务器均可运行



## 准备工作

* 主要是安装 docker, 群晖用户直接到 nas 里面下载套件就行了

* 下载终端工具MobaXterm[点我下载](https://sansanpipi.lanzoul.com/iyTDU0r40hva)

## 第一步 docker 安装
更新源

sudo apt-get update

安装 docker

sudo apt install docker -y

sudo apt install docker.io -y

## 第二步 设 docker 组

sudo groupadd docker

sudo usermod -aG docker $USER

newgrp docker

## 第三步 开放端口

nas 用户不用管,云服务器的去网页开启所有端口（网上有教程去搜）

## 第四步 镜像拉取

开启管理员

sudo -i

拉取镜像

docker pull ymx123/amd64:latest   （x86镜像）

docker pull ymx123/ymx:latest     （arm镜像）

## 第五步 部署

x86部署

docker run --name study_xxqg -d --privileged=true -p 1234:8080 -v /etc/study_xxqg/:/opt/config/ ymx123/amd64:latest

arm部署

docker run --name study_xxqg -d --privileged=true -p 1234:8080 -v /etc/study_xxqg/:/opt/config/ ymx123/ymx:latest

## 第六步 设置自动重启

docker update --restart=always study_xxqg

## 微信推送

微信推送条件: 

* 云服务器

* 如果是 nas 需要有公网 ip ,或者用内网穿透

## 开启微信号

到 http://mp.weixin.qq.com/debug/cgi-bin/sandbox?t=sandbox/login 手机微信扫码登录既可

* enable 后面的 false 改成true

* 填入一个token，自己编，例如1234，自己记住这个 Token

## 更新

开启管理员

sudo -i

停止和删除容器

docker stop study_xxqg && docker rm study_xxqg

备份数据到data文件夹

mv -f /etc/study_xxqg/* /data

进入etc文件夹

cd /etc

删除study_xxqg文件夹

rm -rf study_xxqg

删除所有带xxqg的镜像

docker rmi --force $(docker images | grep xxqg | awk '{print $3}')

重新拉取镜像

docker run –name study_xxqg -d –privileged=true -p 1234:8080 -v /etc/study_xxqg/:/opt/config/ ymx123/amd64:latest

恢复备份数据回study_xxqg文件夹内

mv -f /data/* /etc/study_xxqg

设置重启

docker update --restart=always study_xxqg
